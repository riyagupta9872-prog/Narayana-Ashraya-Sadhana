What does this app track?

🌙Sleep (Nidra) time
🌅Wake-up time
📿Chanting (Japa) time
📖Reading (Pathan) minutes
🎧Hearing (class/lecture) minutes
🎵Instrument / Activity minutes
😴Day sleep duration
🛠️Service (Seva) — weekly pool
📝Notes Revision — Level-4 only
👗Vaishnav Dress / Tilak / Mala — Sundays

Four Devotee Levels

Your level is assigned by your Admin — it determines your scoring rules and daily maximum.

LevelDaily MaxWeekly Max
Level-1105 pts760 pts
Level-2110 pts795 pts
Level-3115 pts830 pts
Level-4140 pts1005 pts
📌 New devotees start at Level-1 by default.
📌 Level is changed by Team Leader / Dept Admin / Super Admin.

📐 How Weekly % is Calculated

Weekly % = Total Score ÷ (Daily Max × Submitted Days) × 100

Only submitted days count in the denominator — future days are excluded. This is the Fair Denominator system.

📝 How to Fill Daily Sadhana

1️⃣Go to Daily Entry tab on the dashboard.
2️⃣Select the date (today or a missed past date).
3️⃣Fill in all the fields listed below.
4️⃣Tap Submit Sadhana ✅

🔑 First Super Admin Setup

Go to Firebase Console → Firestore → users → your document
Set field: role = superAdmin (case-sensitive)
Can I fill yesterday's sadhana?
Yes! Select yesterday's date from the date dropdown in Daily Entry.
Why is my weekly % lower than expected?
NR days count as −30 and are included in the denominator. Check if you have any missed days.
What is NR?
Not Reported — if you don't fill sadhana for a day, it's recorded as NR with −30 penalty.
Why no marks in the Seva column?
Service is a weekly pool — marks are calculated on Saturday. Daily table shows minutes only. Score appears in the week header.
Can bonus % exceed 100%?
Yes! Bonus marks (instrument, dress, notes) are not added to the denominator, so your % can go above 100%.
I can't see Dress / Tilak / Mala fields.
These fields appear only on Sundays.
My level changed — will old scores change?
No. Old entries keep their original scores. Only new entries use the updated level.
How do I change my password?
Dashboard → Profile menu → Change Password.
